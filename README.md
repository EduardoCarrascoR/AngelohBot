
# AngelohBot

Bot Angeloh post memes in channel discord, this bot is a parody with friend


## Commands server

Install packages

```bash
  npm install
```

Run server bot

```bash
  npm run start
```

Update SlashCommands bot

```bash
  npm run commands
```

## Function in Discord

Meme random 

```bash
  /ramdon_meme
```

Ping

```bash
  /ping
```
